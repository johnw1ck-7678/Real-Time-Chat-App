# Real-Time Chat App

A real-time chat application with instant messaging.

## Features
- Real-time messaging
- Anonymous users.
- Chat rooms / private chats
- Responsive design

## Tech Stack
- Frontend: HTML,React JS
- Backend: Spring Boot
- Database: MongoDB

## Installation
1. Clone the repository.
2. Install dependencies.
3. Run the development server.

## Usage
Start the server and open the app in your browser/device.
